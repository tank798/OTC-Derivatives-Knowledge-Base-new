# 项目文档入口

日常最常用的文件都放在这里：

- [场外衍生品法规知识库](场外衍生品法规知识库.html)：按发文主体、效力状态、业务分类和格式查看114份法规及全部引用单元。
- [项目迭代记录](项目迭代记录.md)：记录每次版本为什么改、改了什么、如何验证；UI 变化配有截图。
- [项目整体逻辑说明](architecture/项目整体逻辑说明.md)：面向业务人员说明从法规原件、Chunk、混合检索到回答的完整逻辑。
- [问答与检索架构](architecture/问答与检索架构.md)：说明 Agent、检索、证据合并、引用校验和 Wiki 的边界。
- [Chunk 复核报告](reviews/Chunk复核报告.md)：保留全量 Chunk 复核后的压缩结论。
- [专家知识 Wiki](../wiki/专家知识Wiki.md)：存放用户确认的业务 Know-how；它不能替代法规原文。

正式数据仍保留在 `data/`：原始监管文件、结构化 JSON、正式 Chunk 和 BM25/向量索引都没有移动。

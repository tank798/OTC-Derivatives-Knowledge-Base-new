# 问答验证集说明

本目录中的问题只用于离线验证，不参与运行时 Prompt、召回排序、法规筛选或回答生成。

- `queries.jsonl`：用户最初提供的3个真实回归问题，用于观察旧问题是否因通用改动退化。
- `general_queries.jsonl`：覆盖利率、期权、保证金、商业银行、基金估值等主题的通用问题，用于检查改动是否只对少数样例有效。
- `live-results/`：真实 API 运行结果和耗时，已被 Git 忽略。
- `logs/`：Agent 逐步运行日志，已被 Git 忽略。

`live-agent-smoke.mjs` 的“通过”只表示对话链路完成、引用 ID 存在且原文真实性校验通过，不代表法律结论已经得到人工专业审核。回答质量需要用冻结问题集进行盲评，且人工标注不得进入运行时代码。

手动核验时，应按真实用户流程观察问题改写并确认或纠正，再阅读最终回答、右侧完整 Chunk 和官网链接；必要时查看 `logs/` 中的检索轮次、保留证据、引用校验和模型调用耗时。三道回归问题不拥有专项代码或专项 Prompt。
